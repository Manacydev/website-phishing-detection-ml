🛡️ Safety Tips

  • Check the domain: Phishing sites use misspellings — G00gle vs Google.
  • Look for HTTPS: Legitimate sites use secure connections (🔒 padlock).
  • Hover before clicking: See the real destination before you click any link.
  • Beware of urgency: "Act now!" messages are classic phishing pressure tactics.
  • Protect credentials: Never enter passwords on suspicious pages.

🤖 How PhishGuard Works

PhishGuard uses a Random Forest classifier trained on 30 security features extracted from URLs and their web content.

When you submit a URL, the system extracts features in real time — checking URL structure, WHOIS data, page content, form handlers, scripts, and more. The model then predicts whether the site is safe or a phishing attempt, along with a confidence score.
